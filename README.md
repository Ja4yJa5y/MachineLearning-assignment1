# MachineLearning-assignment1
Dataset Description: 
Dataset Source: The dataset is sourced from the Google Play Store Apps repository (Kaggle), containing details on approximately 10,000 apps.
Dimensions: * Original: 10,841 rows × 13 columns.
• Cleaned: 8,196 rows × 11 columns (after deduplication and handling missing values).
Feature Descriptions:
• App: Name of the application.
• Category: The industry or genre (e.g., Family, Game).
• Rating: Average user rating (0.0 to 5.0).
• Reviews: Total number of user reviews.
• Size: Application file size in Megabytes (MB).
• Installs: Total number of user downloads.
• Type: Whether the app is Free or Paid.
• Price: The cost of the app (in USD).
• Content Rating: Target age group (e.g., Everyone, Teen).
• Genres: Detailed sub-category of the app.
• Last Updated: The date of the most recent developer update.
Purpose of the Dataset: To analyze market trends within the Android ecosystem, identify factors that influence high user ratings, and understand the relationship between app maintenance (updates) and popularity.
